Easily add an online shopping cart to your blog. This plugin will create a store for you and automatically link it in your blog.

== Description ==

This plugin will create a store for your on KloudStores (kloudstores.com) and also link it in your blog as a separate page.
After the store is created you can manage it at https://admin.kloudstores.com

KloudStores is a hosted shopping cart solution which allows you to easily customize and manage your online store.
Get product views and order information in real-time.

For more information, check out www.kloudstores.com

The created online store includes features as:

* Product inventory and management
* Order management
* Reports
* Discount codes and shipping management
* Multiple themes and an advanced theme editor
* Facebook integration

== Installation ==

1. Upload the entire `kloudstores` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
